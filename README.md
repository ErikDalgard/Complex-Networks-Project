# Network Analysis Project

Repository structure:

- **project.ipynb** – Main notebook. Run this notebook to generate all figures and results used in the report.
- **info_functions.py** – Contains all functions used to compute network measures and generate the corresponding plots.
- **generate_networks.py** – Generates the artificial networks (Gilbert, Watts-Strogatz, Configuration model and Barabási-Albert).
- **data/** – Contains the GRQC collaboration network dataset.
